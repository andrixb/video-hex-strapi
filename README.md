# Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000/trivia](http://localhost:3000/trivia) with your browser to see the result.

## Test
### For unit testing
`yarn jest` 

### For e2e tests
In two separates terminals:
`yarn cypress` 
and
`yarn dev`
